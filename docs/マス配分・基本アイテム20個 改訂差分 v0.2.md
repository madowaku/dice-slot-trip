\# 6. アイテムマスの新抽選



アイテムマス停止時は、最初に報酬カテゴリを抽選する。



| 報酬カテゴリ      |  確率 |

| ----------- | --: |

| DICE\_ADD\_1  | 35% |

| 通常アイテム1個    | 55% |

| 2候補からアイテム1個 | 10% |



\## DICE\_ADD\_1



現在ダイス数を1増加する。



最大3。



3ダイス状態で当選した場合は、12コインへ変換する。



\## DOUBLEで停止した場合



2ダイスのDOUBLEでアイテムマスへ停止した場合：



1\. DICE\_ADD\_1を確定

2\. 通常アイテム抽選を1回行う



これにより、DOUBLE成功時は3ダイスへ進みつつ、アイテムも獲得できる。



\## 3ダイス役との接続



| 役        | アイテムマス効果     |

| -------- | ------------ |

| TRIPLE   | RARE抽選を含む2候補 |

| STRAIGHT | 移動・ダイス系候補を優先 |

| PAIR     | 2候補から1個      |

| ALL EVEN | 保険系候補を優先     |

| ALL ODD  | 振れ幅のあるアイテム候補 |

| 役なし      | 基本抽選         |

\# 11. ダイス数決定の処理順



停止マスから追加ダイスを得た場合、次回ダイス数は以下の順で決定する。



1\. 今回のロール結果から、基本の次回ダイス数を決める

2\. DICE\_KEEPなどの保持効果を適用

3\. 停止マスの効果を解決

4\. DICE\_ADD\_1などの獲得を加算

5\. 最大3に制限

6\. 余剰分を12コインへ変換

7\. 次回ダイス数を保存



例：



```text

2ダイスでDOUBLE失敗

基本の次回ダイス数：1

アイテムマスでDICE\_ADD\_1

最終的な次回ダイス数：2

```



例：



```text

3ダイスでPAIR

基本の次回ダイス数：2

アイテムマスでDICE\_ADD\_1

最終的な次回ダイス数：3

```
