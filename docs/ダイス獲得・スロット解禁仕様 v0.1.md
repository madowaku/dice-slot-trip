\# Dice Slot Trip



\## ダイス獲得・スロット解禁仕様 v0.1



\# 1. 本仕様の目的



本作のダイスシステムを、次の構造へ変更する。



> 普段は1ダイスで静かに旅をする。

> イベントやマスから追加ダイスを獲得する。

> 2ダイスではDOUBLEを狙う。

> 3ダイスがそろうとDice Slotが解禁される。

> スロット役の結果を楽しんだ後、基本の1ダイスへ戻る。



3ダイスを常時使用するのではなく、\*\*旅の途中で到達する短い特別状態\*\*として扱う。



この変更により、以下を実現する。



\* 通常時とスロット時の緩急

\* 追加ダイス獲得そのものの報酬化

\* 目押し操作の目的変化

\* PAIRやTRIPLEの特別感維持

\* アイテムマス、イベントマス、リスクマスの価値向上

\* タイトル「Dice Slot Trip」とゲーム体験の一致



\---

\# 3. ダイス状態



通常のダイス状態は3段階とする。



| 状態ID         | ダイス数 | 状態名       | 主な目的      |

| ------------ | ---: | --------- | --------- |

| NORMAL       |    1 | 通常旅行      | 停止マスを狙う   |

| DOUBLE\_READY |    2 | ダブルチャンス   | DOUBLEを狙う |

| SLOT\_READY   |    3 | Dice Slot | 複数の役を狙う   |



5ダイスは通常状態には含めず、イベントなどで一時的に発生する。



```text

FESTIVAL = 5ダイス

```



\---



\# 4. 現在ダイス数



ゲーム状態として以下を保持する。



```text

current\_dice\_count

```



有効値：



```text

1

2

3

```



通常時の最小値は1、最大値は3。



5ダイスイベント中も`current\_dice\_count`は変更せず、イベント専用値として管理する。



```text

temporary\_roll\_dice\_count = 5

```



\---



\# 5. 旅開始時



新しい旅を開始した場合、ダイス数は1とする。



```text

current\_dice\_count = 1

```



対象：



\* ステージ選択から出発

\* ボスゲット後に「次の旅へ」を選択

\* 新規ゲーム開始

\* DEBUGから旅をリセット



アプリ再起動や「つづきから」では初期化せず、保存されたダイス数を復元する。

\# 6. 1ダイス状態



\## 6.1 役割



通常プレイの中心となる状態。



全ロールの55〜65%程度を目標とする。



\## 6.2 動作



\* サイコロを1個振る

\* 出目分だけ移動する

\* スロット役判定は行わない

\* 早止め操作は利用可能

\* 停止マスの効果を通常どおり発動する



\## 6.3 プレイ上の目的



1ダイスでは、数字そのものよりも停止位置を狙う。



例：



\* 3ならイベントマス

\* 4ならアイテムマス

\* 5ならリスクマス

\* 6ならボス気配マス



目押しが成功を保証しない既存方針は維持する。



\## 6.4 ロール後



追加ダイス獲得がなければ、次回も1ダイス。

\# 7. 2ダイス状態



\## 7.1 状態名



画面上の表示名：



\*\*DOUBLE CHANCE\*\*



常時表示ではなく、2ダイス到達時に短く表示する。



\## 7.2 動作



\* サイコロを2個振る

\* 合計値分だけ移動する

\* 3ダイス用の役は判定しない

\* 2個が同じ数字の場合のみDOUBLE成立

\* 早止め操作は左から順に利用可能



\## 7.3 DOUBLE



例：



```text

2・2

4・4

6・6

```



同じ数字ならDOUBLE成立。



\### DOUBLE基本効果



\* 停止マス効果を少し強化

\* 追加ダイスを1個獲得

\* 次回のダイス数を3へ増加

\* 「DICE SLOT READY」を表示



つまりDOUBLEは、2ダイスから3ダイスへ進む主な方法となる。



\### 停止マス効果の強化例



| マス   | DOUBLE効果       |

| ---- | -------------- |

| 通常   | ボス気配＋1         |

| イベント | イベント報酬に小ボーナス   |

| アイテム | 候補を2個表示        |

| コイン  | 獲得量＋50%        |

| ボス気配 | 気配上昇量＋1        |

| リスク  | 失敗時の軽い不利益を一度防ぐ |

| 名所   | 追加の旅メモ抽選       |

| 休憩   | 小さなコイン報酬を追加    |



すべてのマスに専用効果を実装する必要はない。



効果が未定義の場合は、共通効果として以下を適用する。



```text

BOSS\_SCENT +1

```



\## 7.4 DOUBLE不成立



DOUBLEが成立しなかった場合、ロール後に1ダイスへ戻る。



```text

current\_dice\_count = 1

```



\## 7.5 2ダイス時の追加ダイス獲得



DOUBLE不成立でも、停止マスやイベント結果から追加ダイスを獲得した場合は、その獲得を反映する。



例：



```text

2ダイスで役なし

→ 通常なら1ダイスへ戻る

→ アイテムマスで追加ダイス＋1

→ 次回は2ダイス

```



消費後の基本値へ戻してから、獲得分を加える。

\## 8.1 状態名

\## 8.3 対象役

\* STRAIGHT

\* ALL ODD

\* ALL EVEN



複数成立時の優先順位は既存仕様を維持する。



```text

ALL ODD

ALL EVEN

```



必要なイベントでは複数役を参照してもよい。

\# 9. 3ダイス後の状態変化



3ダイスは原則として一度のロールで消費し、1ダイスへ戻る。



ただし成立役によって、追加ダイスを保持または再獲得できる。



| 結果         | 次回ダイス数 | 補足          |

| ---------- | -----: | ----------- |

| TRIPLE     |      3 | Dice Slot継続 |

| STRAIGHT   |      2 | 追加ダイス1個獲得   |

| PAIR       |      2 | 追加ダイス1個獲得   |

| ALL EVENのみ |      2 | 安定継続        |

| ALL ODDのみ  |      1 | 報酬強化、継続なし   |

| 役なし        |      1 | 通常旅行へ戻る     |



\## 9.1 TRIPLE



TRIPLE成立時：



\* ボス遭遇確定

\* 次回も3ダイス

\* 「DICE SLOT CONTINUE」を表示



TRIPLEによる3ダイス継続は1回までに制限しない。



ただし長い連鎖が発生した場合はプレイログで監視する。



\## 9.2 STRAIGHT



STRAIGHT成立時：



\* ボス気配＋1

\* 次回2ダイス

\* イベントやワープとの既存接続を維持



\## 9.3 PAIR



PAIR成立時：



\* 停止マスまたは交流に小ボーナス

\* 次回2ダイス



\## 9.4 ALL EVEN



上位役が成立していない場合のみ、ALL EVENの継続効果を使う。



\* 次回2ダイス

\* 軽い不利益を受けにくい

\* 安全な旅の流れを維持する



\## 9.5 ALL ODD



上位役が成立していない場合のみ適用。



\* 次回1ダイス

\* 現在の報酬やイベント結果を強化

\* リスクマスでは大当たりと空振りの幅を増やす



ALL ODDは「現在の勝負」、ALL EVENは「次回への継続」という性格に分ける。

\# 10. 追加ダイス獲得



\## 10.1 基本処理



追加ダイスを獲得した場合：



```text

current\_dice\_count += gained\_dice

```



最大3。



```text

current\_dice\_count = clamp(current\_dice\_count, 1, 3)

```



\## 10.2 獲得単位



基本は1個。



```text

DICE\_ADD\_1

```



レア報酬のみ、一気に3ダイス状態へ変更できる。



```text

DICE\_SLOT\_READY

```



\## 10.3 余剰獲得



3ダイス状態で追加ダイスを獲得した場合、余剰分は「ダイスストック」へ変換しない。



代わりに以下へ変換する。



```text

COIN\_S = 12コイン

```



ただし「次のスロットを保持する」効果を持つ専用アイテムは別扱いとする。



\## 10.4 獲得演出



ダイス表示枠を常に3枠用意する。



1ダイス：



```text

\[🎲] \[＋] \[＋]

```



2ダイス：



```text

\[🎲] \[🎲] \[＋]

```



3ダイス：



```text

\[🎲] \[🎲] \[🎲]

DICE SLOT READY

```



追加ダイス獲得時は、空き枠へサイコロが入る短い演出を行う。



演出時間目安：



```text

0.3〜0.6秒

```



スキップ可能にしてよい。

| ------ | ---------: |

| アイテムマス |          高 |

\## 11.1 アイテムマス



基本抽選に追加ダイスを含める。



推奨結果：



| 結果      |  確率 |

| ------- | --: |

| 追加ダイス＋1 | 35% |

| 通常アイテム  | 55% |

| アイテム2候補 | 10% |



DOUBLEで停止した場合：



\* 追加ダイス＋1を確定

\* さらに通常アイテム抽選を行う



\## 11.2 イベントマス



イベント報酬として以下を利用する。



```text

DICE\_ADD\_1

DICE\_SLOT\_READY

DICE\_KEEP

```



全イベントで獲得させず、30イベント中8〜12本程度に設定する。



\## 11.3 リスクマス



安全策：



\* 小報酬

\* または追加ダイス＋1



挑戦策：



\* 成功で追加ダイス＋1

\* 大成功で3ダイス状態へ即時移行

\* 失敗でも現在ダイスを失わない



リスクマスはダイスを集める主要経路の一つとする。



\## 11.4 周回ゲート



周回ゲート通過時、現在ダイス数が1の場合のみ、低確率で追加ダイス＋1。



初期確率：



```text

30%

```



2または3ダイスの場合は追加しない。



周回ゲートで毎回確定獲得にはしない。

\# 13. ダイス保持効果



\## 13.1 DICE\_KEEP



次のロール後に発生するダイス数減少を一度だけ無効化する。



例：



```text

3ダイスで役なし

通常：1ダイスへ戻る

DICE\_KEEPあり：3ダイスを維持

```



効果発動後に消費する。



TRIPLEなどで自然に3ダイスを維持できた場合、DICE\_KEEPは消費しない。



\## 13.2 重複



DICE\_KEEPは1回分だけ保持可能。



すでに保持中に再獲得した場合：



```text

COIN\_Sへ変換

```

\# 14. 5ダイス状態



\## 14.1 位置づけ



5ダイスは通常の成長段階ではなく、低頻度のお祭り状態。



画面表示名：



\*\*DICE FESTIVAL\*\*



\## 14.2 発生源



\* 砂丘のサイコロ祭り

\* リスクマス大成功

\* レアアイテム

\* ボスゲット後ボーナス

\* 将来の期間限定イベント



通常UIから自由に選択できない。



\## 14.3 動作



\* 5個のダイスを振る

\* イベント指定に応じて結果を解決する

\* 移動する場合と移動しない場合をイベント側で指定する

\* 早止め操作を利用可能

\* 成立した役の種類数を参照可能

\* FIVE\_OF\_A\_KINDを判定可能



\## 14.4 終了後



5ダイスイベント終了後は、イベント開始前の`current\_dice\_count`へ戻る。



5ダイス結果によって追加ダイス報酬を得た場合のみ加算する。

\# 17. 目押しの役割



\## 17.1 1ダイス



停止したいマスの数字を狙う。



\## 17.2 2ダイス



最初の数字を見て、2個目を同じ数字に止めてDOUBLEを狙う。



例：



```text

\[4] \[?]

```



4を狙う理由が明確になる。



\## 17.3 3ダイス



途中結果を見て狙いを切り替える。



例：



```text

\[5] \[5] \[?]

```



\* 5を狙ってTRIPLE

\* 4または6を狙ってSTRAIGHTの可能性

\* 奇数を狙ってALL ODD

\* 偶数へ逃げて役なしを避ける



\## 17.4 目押し精度



目押しは結果を保証しない。



\* ダイス表示は高速で切り替わる

\* タップ後に短い停止揺らぎがある

\* 集中すると成功率が少し上がる

\* 数字指定能力にはしない



リスクマスでは一時的に早止め受付を広げる補助を提示できる。

低確率で、通常マスから追加ダイスを発見する。



初期確率：



```text

5%

```



イベント増加と重なってテンポを壊さないよう、一行通知で処理する。

\# 22. ボス交流との接続



\## TRIPLE



既存どおりボス遭遇確定。



さらに次回も3ダイスを維持する。



\## PAIR



既存どおり交流ゲージへ小ボーナス。



次回2ダイス。



\## DOUBLE



DOUBLEで停止したターンにボス遭遇した場合：



\* 交流ゲージ＋2

\* 3ダイス状態へ進む



\## ボスゲット報酬



ボスゲット後、次個体の旅を開始する際は1ダイスへ戻す。



ただし低確率で「門出のサイコロ」を受け取り、2ダイスから始められる。



初期確率：



```text

20%

```



図鑑登録そのものを主報酬とし、確定では与えない。

\# 24. SE方針



\## ダイス獲得



柔らかい小さな着地音。



```text

ころん

```



\## 2ダイス到達



低い音＋少し高い音の2音。



\## 3ダイス到達



短い3音。



3音目だけ少し余韻を長くする。



\## DOUBLE



同じ音を短く2回。



\## Dice Slot終了



減衰する小さな音。



失敗音にはせず、通常旅行へ戻る合図とする。

\# 26. 旧セーブ移行



旧セーブから読み込む場合：



\## 旧選択が1ダイス



```text

current\_dice\_count = 1

```



\## 旧選択が3ダイス



```text

current\_dice\_count = 2

```



既存プレイヤーへの小さな移行ボーナスとして扱う。



\## 旧選択が5ダイス



```text

current\_dice\_count = 3

```



旧5ダイス予約は破棄し、3ダイスへ変換する。



\## 旧チャージ



```text

one\_die\_charge

five\_dice\_charge

```



新システムでは使用しない。



必要なら旧チャージの合計が1以上の場合、移行時に追加ダイス＋1を一度だけ付与する。



移行処理は重複実行しない。
