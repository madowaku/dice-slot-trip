\# Dice Slot Trip



\## 基本アイテム20個 v0.1



\## 1. 基本方針



アイテムは、プレイヤーへ複雑な攻略を要求するものではなく、普段のサイコロ旅に小さな選択を加えるために使用する。



設計原則は以下。



\* 使用しなくてもステージを進められる

\* 役を無条件で確定させるアイテムは基本的に作らない

\* 使用結果が直感的に分かる

\* 1回の使用で長い設定画面を開かない

\* 「今使うか、次まで残すか」で少し迷える

\* ボスゲットに必須のアイテムは作らない

\* ステージ間で共通利用できる

\* アイテムを抱え込むより、適度に使った方が楽しいバランスにする



\---



\# 2. 所持ルール



\## 所持枠



初期所持上限は\*\*4枠\*\*とする。



同じアイテムは1枠に最大3個まで重ねられる。



所持上限を超えて獲得した場合は、以下から選択する。



\* 既存アイテムと交換

\* 獲得アイテムをその場で使用

\* 獲得アイテムをコインへ変換

\* 獲得をキャンセル



\## 使用タイミング



アイテムは以下の種類に分ける。



| 種類    | 使用タイミング           |

| ----- | ----------------- |

| 即時型   | アイテム画面から使用するとすぐ発動 |

| ロール前型 | サイコロを振る前に使用       |

| ロール後型 | 出目確定後、移動前に使用      |

| 自動型   | 条件を満たしたとき自動発動     |

| 遭遇前型  | ボス交流画面へ入る前に使用     |



\## 同時使用制限



1回の移動で使用できるアイテムは原則として以下。



\* ダイス数変更：1個まで

\* 出目操作：1個まで

\* 移動補助：1個まで

\* 自動保険：条件を満たせば1個



「1ダイス化」と「5ダイス化」は同時に使用できない。

\# 5. 移動・ダイス数変更アイテム



\## ITEM-01「ワープストーン」



\*\*分類：\*\* 移動

\*\*レアリティ：\*\* COMMON

\*\*使用タイミング：\*\* 即時型



\### 効果



現在地から6マス先へ移動する。



ワープ先のマス効果は発動しない。



\### 使用不可条件



\* イベントモーダル表示中

\* ボス交流中

\* 移動演出中

\* 周回ゲート処理中



\### 役割



イベントマスやボス気配マスへ近づくためではなく、周回速度や位置を軽く変える基本アイテム。



\---



\## ITEM-02「折りたたみ自転車」



\*\*分類：\*\* 移動

\*\*レアリティ：\*\* COMMON

\*\*使用タイミング：\*\* ロール前型



\### 効果



次のサイコロ移動距離に＋4マスする。



サイコロの出目と役判定には影響しない。



\### 補足



ワープではないため、最終停止マスの効果は通常どおり発動する。



\### 表示文



> 荷物から、小さな自転車を組み立てた。



\---



\## ITEM-03「のんびり切符」



\*\*分類：\*\* ダイス数変更

\*\*レアリティ：\*\* UNCOMMON

\*\*使用タイミング：\*\* ロール前型



\### 効果



次のロールを1ダイスへ変更する。



出目確定後、移動前に\*\*0〜2マスの範囲で追加移動\*\*を選択できる。



例：



\* 出目3の場合、3〜5マスから選択可能



\### 役割



近くの特殊マスを狙うための精密移動。



\### 制限



\* 役判定は発生しない

\* 5ダイス変更と併用不可

\* 追加移動で周回ゲートを越えることは可能



\---



\## ITEM-04「サイコロ祭り券」



\*\*分類：\*\* ダイス数変更

\*\*レアリティ：\*\* UNCOMMON

\*\*使用タイミング：\*\* ロール前型



\### 効果



次のロールを5ダイスへ変更する。



移動距離、役判定、早止め操作は既存の5ダイス仕様に従う。



\### 役割



狙ったマスへ止まるより、複数役や大きな移動を期待する大振りアイテム。



\### 演出



使用時のみ、サイコロを置く台へ小さな祭り旗を表示する。



音量や発光は通常の落ち着いた演出を維持する。

\# 6. 出目・役補助アイテム



\## ITEM-05「旅人の予備ダイス」



\*\*分類：\*\* 出目操作

\*\*レアリティ：\*\* COMMON

\*\*使用タイミング：\*\* ロール後型



\### 効果



出目確定後、サイコロを1個選んで一度だけ振り直す。



振り直した後に役を再判定する。



\### 制限



\* 振り直した目を元へ戻せない

\* 追加イベントダイスでも使用可能

\* 1ダイス時は使用不可



\### 役割



ペアからトリプルを狙う、ストレートの欠けた目を狙うなど、軽い勝負を作る。



\---



\## ITEM-06「ホールドクリップ」



\*\*分類：\*\* 出目操作

\*\*レアリティ：\*\* UNCOMMON

\*\*使用タイミング：\*\* ロール後型



\### 効果



サイコロを1個選んで保持し、残りのサイコロを一度だけ振り直す。



例：



```text

5・5・2

```



5を1個保持した場合、残りの5と2を振り直す。



\### 制限



\* 5ダイス時も保持できるのは1個

\* 振り直し後に役を再判定

\* 予備ダイスとの同時使用不可



\### 役割



現在の小当たりを捨て、より大きな役を狙う判断を作る。



\---



\## ITEM-07「目盛り調整工具」



\*\*分類：\*\* 出目操作

\*\*レアリティ：\*\* RARE

\*\*使用タイミング：\*\* ロール後型



\### 効果



サイコロ1個の出目を＋1または－1できる。



\### 制限



\* 1から－1は不可

\* 6から＋1は不可

\* 1回のロールにつき1個のみ

\* 使用後に役を再判定

\* 1ダイス時も使用可能



\### 例



```text

2・4・5

```



2を3へ変更すると、3・4・5のSTRAIGHTが成立する。



\### 役割



役を完全に保証せず、「あと一歩」の出目を救うレアアイテム。



\---



\## ITEM-08「そろいのリボン」



\*\*分類：\*\* 役補助

\*\*レアリティ：\*\* COMMON

\*\*使用タイミング：\*\* ロール前型



\### 効果



次にPAIRが成立したとき、そのPAIRによるマスまたはイベント報酬を1.5倍相当に強化する。



PAIRが成立するまで保持される。



\### 対象例



\* コイン獲得量増加

\* アイテム候補数増加

\* ボス交流ボーナス増加

\* ショップ割引量増加



\### 対象外



\* TRIPLE

\* STRAIGHT

\* ALL ODD

\* ALL EVEN

\* 基本移動距離

\## ITEM-10「冷たいミントティー」



\*\*分類：\*\* ボス交流

\*\*レアリティ：\*\* COMMON

\*\*使用タイミング：\*\* 遭遇前型



\### 効果



次のボス交流で得られる交流ゲージに＋4する。



効果は次の交流まで保持される。



\### 補足



個体の好みに関係なく使用可能。



スフィンクス以外のボスでは、ステージごとに見た目と名称を差し替えてもよい。

\## ITEM-11「デーツの小袋」



\*\*分類：\*\* ボス交流

\*\*レアリティ：\*\* UNCOMMON

\*\*使用タイミング：\*\* 遭遇前型



\### 効果



次の交流画面で、現在個体が好む選択肢へ小さなアイコンを表示する。



好みの選択肢を選んだ場合、交流ゲージに追加で＋4する。



\### 表示例



```text

静かに隣へ座る　♡

元気よく挨拶する

```



\### 設計意図



正解を知らなくても交流は進むが、アイテムを使うと性格を読みやすくなる。

\## ITEM-12「旅先からの絵葉書」

\*\*レアリティ：\*\* RARE

\*\*使用タイミング：\*\* 遭遇前型

\### 制限

\* シンガポールのスコール

\* 東京の強風

\* パリの濃霧



\### 対象外



区域内の報酬増加効果は失わない。

\# 11. ショップ価格記号

| 記号      | 想定               |

| ------- | ---------------- |

| PRICE\_S | 通常プレイ数ターンで入手できる額 |

\* UNCOMMON：PRICE\_M

\# 15. DEBUG・テスト項目



以下をDEBUGから確認できるようにする。



\* 指定アイテムを付与

\* 所持数変更

\* 所持上限変更

\* 次回アイテム抽選を固定

\* アイテム効果を即時解除

\* RAREアイテム抽選

\* 全アイテム削除



最低限の検証項目：



1\. 所持上限を超えた場合に交換画面が出る

2\. 同一アイテムが最大3個まで重なる

3\. 1ダイスと5ダイスを同時使用できない

4\. 出目操作後に役が再判定される

5\. 移動補助が役判定へ影響しない

6\. ワープ先のマス効果が連鎖しない

7\. ボス交流補助が一度だけ消費される

8\. 自動保険が対象条件でのみ発動する

9\. 保存後もアイテムと予約効果が復元される

10\. 図鑑アイテムで重複登録されない

11\. 追加イベントダイスでも対応アイテムを使用できる

12\. 使用不能時に理由が表示される
