\# Dice Slot Trip



\## 名所発展・景観成長仕様 v0.1



\# 1. 目的



名所マスへ停止するたびに、周辺の建物、屋台、料理、人々、装飾が増え、ステージの景観が少しずつ成長するシステムを実装する。



本システムにより、以下を実現する。



\* 一度通り過ぎた名所を、次の周回で狙いたくなる

\* 同じ90マスを周回しても、景色が変化し続ける

\* 1ダイスの目押しや移動アイテムに新しい目的が生まれる

\* 周回ポイント以外にも、目に見える進行を与える

\* 建物、料理、風景のコレクション欲を刺激する

\* 名所完成後も、再訪が無駄にならない

\* ステージ全体がプレイヤー自身の旅によって華やかになる



名所発展は周回効率を妨げる長いイベントにはせず、短い演出と確実な報酬で処理する。



\---



\# 2. コア体験



基本循環：



```text

名所を通り過ぎる

↓

次周で止まりたい目標として記憶する

↓

1ダイス、ぴったり切符、移動アイテムで狙う

↓

名所へ停止

↓

建物や屋台が増える

↓

ご当地コレクションを獲得

↓

盤面が華やかになる

↓

次の発展段階や別の名所を狙う

```



名所は、危険マスの「避けたい」に対する、強い「止まりたい」を担当する。



\---



\# 3. 名所数



カイロステージには、既存配分どおり3つの名所マスを配置する。



| ID              | 地区    | 名所名      |

| --------------- | ----- | -------- |

| CAI\_LANDMARK\_01 | 市場    | 香辛料市場通り  |

| CAI\_LANDMARK\_02 | ピラミッド | 夕映えの展望広場 |

| CAI\_LANDMARK\_03 | 遺跡    | 星灯りの遺跡広場 |



名所マス数は3のまま変更しない。



\---



\# 4. 名所発展レベル



各名所は4状態を持つ。



| レベル | 状態   | 内容                |

| --: | ---- | ----------------- |

|   0 | 未訪問  | 名所の輪郭や空き地だけが見える   |

|   1 | 発見   | 基本となる建物や設備が出現     |

|   2 | にぎわい | 屋台、料理、人、動物などが増える  |

|   3 | 完成   | 装飾、照明、特別な風景が追加される |



内部値：



```text

landmark\_level = 0〜3

```



名所へ停止すると、未完成の場合は必ず1段階発展する。



抽選による失敗はない。



\---



\# 5. 発展処理



名所へ停止した場合：



```text

if landmark\_level < 3:

&#x20;   landmark\_level += 1

&#x20;   発展演出

&#x20;   発展報酬

&#x20;   コレクション登録

else:

&#x20;   再訪処理

```



1回の停止で2段階以上発展しない。



PAIR、TRIPLE、アイテム効果などによって発展量を倍増させない。



名所完成までには、その名所へ合計3回停止する必要がある。



\---



\# 6. 共通発展報酬



名所へ停止したターンは、既存のラップボーナスを獲得する。



```text

LANDMARK\_LAP\_BONUS = +8

```



発展時は、レベルに応じて追加報酬を与える。



|    到達レベル | 共通報酬                           |

| -------: | ------------------------------ |

|   Lv1 発見 | 20コイン                          |

| Lv2 にぎわい | DICE\_ADD\_1                     |

|   Lv3 完成 | current\_lap\_bonus +30、特別ポストカード |



\## Lv2のDICE\_ADD\_1



\* 現在ダイス数を1増加

\* 最大3

\* 3ダイス時は12コインへ変換

\* 加速行動としてFLOWを増加させる



\## Lv3完成報酬



\* ラップボーナス+30

\* 名所完成ポストカード

\* ステージ発展値+1

\* 短い完成演出



完成時にDICE\_KEEPなどの強力な継続効果は確定付与しない。



\---



\# 7. 名所停止と周回システム



\## クリーンラップ



名所停止によってクリーンラップを失わない。



\## FLOW



名所へ止まっただけではFLOWを増加させない。



次の場合のみFLOW加速行動になる。



\* Lv2報酬でDICE\_ADD\_1を得た

\* 再訪報酬でDICE\_ADD\_1を選んだ

\* 名所固有イベントから前進効果を得た



\## ラップスコア



すべての名所停止でラップボーナス+8。



Lv3完成時は、さらに+30。

\# 9. 盤面の読みやすさ



景観成長によってマスが読みにくくならないよう、以下を守る。



\* 発展物は基本的にルートの外側へ置く

\* プレイヤートークンと重ねない

\* 危険マス、現在地、到達可能範囲を隠さない

\* 当たり判定を持たせない

\* 景観オブジェクトは操作不能

\* マスアイコンより彩度を少し低くする

\* 同一名所の常時表示物は最大5〜7要素程度

\* 小さな動物や人のアニメは1〜2体まで

\* 点滅や強い発光を使用しない



大型11マス現在地表示では発展景観を詳しく表示する。



全体ミニマップでは、建物を直接表示せず、名所アイコンの段階だけを変化させる。



\---



\# 10. ミニマップ表示



名所アイコンには、発展レベルを示す3つの小さな印を表示する。



例：



```text

香辛料市場通り

● ○ ○

```



完成時：



```text

香辛料市場通り

● ● ●

```



未訪問名所もアイコンを表示する。



次周で狙いたい場所として認識できることを優先する。



\---



\# 11. 発展演出



名所発展時の演出は、0.8〜1.5秒を目標とする。



処理例：



1\. カメラを名所へ少し寄せる

2\. 新しい建物や屋台が柔らかく出現

3\. 料理または風景カードを表示

4\. 報酬を表示

5\. 自動的に盤面へ戻る



\## 演出方針



\* 大きな爆発や紙吹雪は使用しない

\* 建物が上から落下する表現は避ける

\* 砂、風、布、ランタンなどで土地らしさを出す

\* タップで短縮可能

\* 「演出短縮」設定時は0.3〜0.5秒にする



\---



\# 12. ステージ発展値



各名所のレベル合計を、ステージ発展値として扱う。



```text

stage\_development

= landmark\_01\_level

\+ landmark\_02\_level

\+ landmark\_03\_level

```



カイロの最大値：



```text

MAX\_STAGE\_DEVELOPMENT = 9

```



表示例：



```text

カイロ景観 6 / 9

```



\---



\# 13. ステージ全体の景観成長



ステージ発展値に応じて、背景や盤面全体へ小さな変化を追加する。



| 発展値 | ステージ変化            |

| --: | ----------------- |

| 0〜2 | 初期状態              |

|   3 | 旅人や看板が少し増える       |

|   6 | 布、ランタン、植物などが増える   |

|   9 | 完成景観、遠景の灯り、特別な空気感 |



\## 発展値3



\* 市場付近に旅人が増える

\* ルート沿いに小さな案内板が出る

\* 一部の空き地に荷物や壺が追加される



達成報酬：



```text

COIN\_M = 30コイン

```



\## 発展値6



\* ランタンや布飾りが増える

\* オアシスや遺跡に植物が追加される

\* 環境音へごく小さな賑わいを追加可能



達成報酬：



```text

DICE\_SLOT\_READY

```



\## 発展値9



\* 全名所完成

\* 遠景に灯りが増える

\* ステージ選択画面のカイロ表示が完成版になる

\* 「砂時計のカイロ完成ポストカード」を登録



達成報酬：



```text

大きな旅箱 ×1

```



ステージ全体の変化は永続する。

1名所につき6個。



| 分類       | 1名所あたり |     全体 |

| -------- | -----: | -----: |

| 名所カード    |      1 |      3 |

| 名物料理     |      2 |      6 |

| 旅の風景     |      2 |      6 |

| 完成ポストカード |      1 |      3 |

| \*\*合計\*\*   |  \*\*6\*\* | \*\*18\*\* |



表示：



```text

カイロ旅図鑑 11 / 18

```

\# 15. コレクションの獲得方法



各名所の6枠は、次の方法で獲得する。



| 枠        | 獲得条件   |

| -------- | ------ |

| 名所カード    | Lv1到達  |

| 名物料理A    | Lv2到達  |

| 旅の風景A    | Lv3到達  |

| 完成ポストカード | Lv3到達  |

| 名物料理B    | 完成後の再訪 |

| 旅の風景B    | 完成後の再訪 |



基本発展だけで4枠。



残り2枠は完成後の再訪で獲得する。



\---



\# 16. 完成後の再訪



Lv3の名所へ停止すると、発展の代わりに再訪スタンプを得る。



```text

landmark\_revisit\_stamps += 1

```



\## 1回目・2回目



以下から1つを選ぶ。



\* 12コイン

\* ボス気配+1

\* current\_lap\_bonus +10



\## 3回目



未登録の再訪コレクションを1つ確定で登録する。



優先順：



1\. 名物料理B

2\. 旅の風景B



登録後、再訪スタンプを0へ戻す。



```text

landmark\_revisit\_stamps = 0

```



\## 再訪コレクション完成後



その名所の6枠がすべて埋まっている場合、3スタンプごとに次から選ぶ。



\* DICE\_ADD\_1

\* current\_lap\_bonus +25

\* 30コイン



これにより、完全収集後も名所停止が無駄にならない。



\---



\# 17. 重複処理



同じコレクションは重複登録しない。



再訪コレクション獲得時は未登録枠を優先するため、抽選による重複は発生しない。



イベントなどから登録済みコレクションを獲得した場合：



```text

current\_lap\_bonus +15

```



へ変換する。



新しい交換用通貨は追加しない。



\---



\# 18. コンプリート報酬



\## 名所単位



1つの名所で6枠すべて登録すると、その名所へ完成マークを付ける。



報酬：



```text

DICE\_KEEP

```



すでにDICE\_KEEP保持中の場合：



```text

COIN\_S = 12コイン

```



\## ステージ単位



カイロの18枠すべてを登録すると、ステージコレクション完成。



報酬：



\* カイロ旅図鑑の特別フレーム

\* ステージ選択画面へ完成バッジ

\* 特別なダイストレイ外見候補

\* current\_lap\_bonus +100



特別ダイストレイは性能差なし。



\---



\# 19. 隠し差分



コンプリート率には含めない、任意の隠し差分を将来追加できる。



例：



\* クリーン連続3以上で名所停止

\* FLOW MAX中に名所停止

\* ALL ODD成立後に名所停止

\* 夜景状態で名所停止

\* スフィンクスゲット後に再訪



隠し差分は図鑑の18枠へ含めない。



100%達成を極端に難しくしないためである。

\# 25. アイテムとの接続



\## ぴったり切符



名所狙いの主要アイテム。



1ダイス結果を±1し、名所へ着地できる。



\## 折りたたみ自転車



移動距離+4によって遠い名所を狙える。



\## ワープストーン



移動先効果が発動しないため、名所へ到着しても発展しない。



ワープストーンによる通過は停止扱いにしない。



\## 旅人の予備ダイス



名所へ止まれる出目を狙い直せる。



\## 目盛り調整工具



2、3ダイス時に合計を調整し、名所停止を狙える。



\## 旅のカメラ



未完成名所へ停止した場合、発展に加えて小さな旅メモを登録する。



完成済み名所では、再訪スタンプ+1を追加で得る。



1停止で追加できる再訪スタンプは最大1。



\---



\# 26. キャラクターとの接続



\## アオ



周回スキルの移動距離-2／0／+2で名所を狙いやすい。



\## ミラ



名所停止時に写真ポイントを獲得。



名所Lv3完成時は写真ポイント+2。



\## ジン



早止めによって名所へ届く出目を狙いやすい。



\## ルル



完成済み名所の再訪時、低確率で追加の旅メモを見つける。



コレクション本体の必要再訪数は短縮しない。

\# 28. 処理順



名所マス停止時：



1\. 名所IDを取得

2\. 重複解決IDを確認

3\. 名所レベルを取得

4\. Lv3未満なら発展

5\. 景観レイヤーを更新

6\. コレクションを登録

7\. 名所報酬を付与

8\. ラップボーナスを加算

9\. ステージ発展値を再計算

10\. ステージ発展節目を判定

11\. コンプリートを判定

12\. 自動保存

\# 35. 完了条件



\* 名所へ停止するたびに盤面が目に見えて成長する

\* 一度通り過ぎた名所を次周で狙いたくなる

\* 建物、料理、人、動物が段階的に増える

\* 完成後も再訪コレクションが残る

\* コレクションは抽選運だけに依存せず完成できる

\* 周回ポイント、クリーンラップ、FLOWを邪魔しない

\* 景観発展とコレクション欲が次の周回目標を生む
