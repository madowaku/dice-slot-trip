\# Dice Slot Trip



\## 周回ポイント・クリーンラップ・加速連鎖仕様 v0.1



\# 1. 目的



本作の長期的な熱中要素を、以下の周回ループによって構築する。



> デメリットマスを避ける

> メリットマスやアイテムから追加ダイスを得る

> DOUBLEやDice Slotで移動を加速する

> クリーンラップを達成する

> 周回ポイントと倍率を積み上げる

> 次の報酬を目指して、もう一周する



会話や長い物語を周回の主目的にはしない。



プレイヤーが盤面を見て、危険な着地点を避け、加速手段をつなぐことを主な判断要素とする。



\---



\# 2. コアコンセプト



本システムの中心は次の4要素。



1\. \*\*周回ポイント\*\*



&#x20;  \* 1周するたびに必ず獲得できる

&#x20;  \* デメリットで減少しない

&#x20;  \* 累積報酬の進行に使用する



2\. \*\*ラップスコア\*\*



&#x20;  \* その周回をどれだけ効率よく進んだかを示す

&#x20;  \* デメリット、ロール数、加速連鎖によって変動する

&#x20;  \* 自己ベストや将来のランキングに使用できる



3\. \*\*クリーンラップ\*\*



&#x20;  \* デメリットを一度も受けずに1周する

&#x20;  \* 連続達成で周回ポイント倍率が成長する



4\. \*\*加速連鎖\*\*



&#x20;  \* メリット、追加ダイス、役、アイテムを連続させる

&#x20;  \* 連鎖中はポイントと移動効率が上がる



周回ポイントとラップスコアを分けることで、以下を両立する。



\* ゆっくり遊ぶ人も必ず報酬へ近づく

\* 効率を考える人は高スコアを狙える

\* デメリットを踏んでもプレイ時間が無駄にならない

\* 上手く避けたときの達成感は残る

\# 4. 周回ポイント



\## 4.1 基本ルール



1周完了時に周回ポイントを獲得する。



基本値：



```text

BASE\_LAP\_POINTS = 100

```



周回中に得たラップボーナスと、クリーン倍率を加えて最終値を計算する。



```text

周回ポイント獲得量

= floor((100 + ラップボーナス) × クリーン倍率)

```



デメリットを受けても、獲得予定の周回ポイントを直接減らさない。



デメリットは以下へ影響する。



\* クリーンラップ失敗

\* クリーン倍率なし

\* FLOW低下または解除

\* ラップスコア減少

\* 軽いゲーム内不利益



\## 4.2 最低保証



どのような周回でも、最低100ポイントを獲得する。



```text

MIN\_LAP\_POINTS = 100

```



長く遊んだ結果がゼロになることはない。

\# 6. クリーンラップ



\## 6.1 成立条件



周回ゲートから次の周回ゲートまでに、デメリット効果を一度も実際に受けなければ成立する。



```text

current\_lap\_clean = true

```



デメリットが適用された時点で：



```text

current\_lap\_clean = false

```



\## 6.2 クリーンを維持する行動



以下はクリーンラップを維持する。



\* 危険マスへ止まらず通過した

\* 危険マスの追加3ダイス挑戦に成功した

\* 旅のお守りで不利益を完全に無効化した

\* 予備の小財布でコイン減少を完全に防いだ

\* ALL EVENによるガードで不利益を完全に防いだ

\* ステージ効果を折りたたみ傘で完全に防いだ



\## 6.3 クリーンを失う行動



以下の場合、クリーンラップを失う。



\* 危険マスのデメリットを受けた

\* 危険マス挑戦に失敗し、デメリットが適用された

\* イベントの軽い不利益を受けた

\* ステージ固有の不利益を受けた



通常のダイス消費はデメリットではない。



以下ではクリーンを失わない。



\* 2ダイス失敗で1ダイスへ戻る

\* 3ダイス役なしで1ダイスへ戻る

\* アイテムを使った

\* ショップでコインを使った

\* ボスとの交流結果が小さかった

\# 7. クリーン連続数と倍率



クリーンラップ達成時、連続数を1増加する。



```text

clean\_streak += 1

```



最大値：



```text

MAX\_CLEAN\_STREAK = 5

```



| クリーン連続数 | 周回ポイント倍率 |

| ------: | -------: |

|       0 |    ×1.00 |

|       1 |    ×1.15 |

|       2 |    ×1.30 |

|       3 |    ×1.50 |

|       4 |    ×1.75 |

|     5以上 |    ×2.00 |



\## 7.1 クリーン失敗時



クリーンでなかった周回では倍率を使用しない。



```text

clean\_multiplier = 1.00

```



連続数は全消去せず、1段階だけ低下する。



```text

clean\_streak = max(clean\_streak - 1, 0)

```



例：



```text

クリーン連続4

→ 危険マスの不利益を受ける

→ その周回は倍率1.00

→ 次周開始時の連続数は3

```



一度の失敗で長期間の積み上げを完全に失わない。

\# 8. クリーン連続報酬



クリーン連続数が特定段階へ到達したとき、追加報酬を与える。



| 到達段階 | 報酬         |

| ---: | ---------- |

|    2 | COIN\_S     |

|    3 | DICE\_ADD\_1 |

|    5 | DICE\_KEEP  |



同じ連続段階の報酬は、その段階へ再到達すれば再度獲得可能。



3ダイス状態でDICE\_ADD\_1を得た場合は12コインへ変換する。



すでにDICE\_KEEP保持中の場合も12コインへ変換する。

\# 9. 危険マス



\## 9.1 既存実装との関係



現在の5個のリスクマスを、ゲーム内表示上は\*\*危険マス\*\*として扱う。



内部のマス種別名や既存コードは、必要がなければ変更しない。



```text

内部：RISK

表示：危険マス

```



危険マスは盤面上で常に確認できる固定マスとする。



プレイヤーは目押し、追加ダイス、移動アイテムを利用して着地を避けられる。



\## 9.2 停止時の選択



危険マスへ停止すると、次から選ぶ。



\### 受け流す



追加ロールを行わず、表示された軽いデメリットを受ける。



処理時間を短くしたいプレイヤー向け。



\### 目押しで突破



追加3ダイスを振る。



\* 現在ダイス数は消費しない

\* 移動しない

\* 早止め可能

\* 役が成立すればデメリットを無効化

\* 役なしならデメリットを受ける



\## 9.3 挑戦結果



| 結果         | 効果                                 |

| ---------- | ---------------------------------- |

| TRIPLE     | デメリット無効、DICE\_SLOT\_READY、ラップボーナス+30 |

| STRAIGHT   | デメリット無効、DICE\_ADD\_1、ラップボーナス+20      |

| PAIR       | デメリット無効、DICE\_ADD\_1、ラップボーナス+12      |

| ALL EVENのみ | デメリット無効、COIN\_S                     |

| ALL ODDのみ  | デメリット無効、ラップボーナス+20                 |

| 役なし        | デメリット適用                            |



挑戦成功は加速行動として扱う。

\# 12. 加速行動



以下のいずれかが発生したターンを加速ターンとする。



\* DICE\_ADD\_1を獲得

\* DICE\_SLOT\_READYを獲得

\* DOUBLE成立

\* 3ダイス役成立

\* 5ダイス役成立

\* ワープ発動

\* 移動距離+3以上の効果を使用

\* 危険マス挑戦成功

\* イベントから前進効果を得る

\* FLOW専用の加速マスへ停止



1ターンに複数条件を満たしても、FLOW増加は1回だけ。



```text

flow\_triggered\_this\_turn = true

```

\# 13. 加速連鎖 FLOW



\## 13.1 基本



加速行動が発生したターン終了時、FLOWを1増加する。



```text

flow\_level += 1

```



最大値：



```text

MAX\_FLOW\_LEVEL = 5

```



画面表示：



```text

FLOW 1

FLOW 2

FLOW 3

FLOW 4

FLOW MAX

```



\## 13.2 加速がなかったターン



加速行動が発生しなかった場合、FLOWを1減少する。



```text

flow\_level = max(flow\_level - 1, 0)

```



一度の静かなターンで全解除しない。



\## 13.3 デメリット適用時



危険マスなどのデメリットを実際に受けた場合、FLOWを0へ戻す。



```text

flow\_level = 0

```



アイテムや役で完全に無効化した場合は解除しない。

\# 14. FLOWボーナス



加速行動を成功させたターン、現在FLOWに応じたラップボーナスを獲得する。



| FLOW | 追加ラップボーナス |

| ---: | --------: |

|    1 |        +5 |

|    2 |       +10 |

|    3 |       +15 |

|    4 |       +20 |

|    5 |       +30 |



FLOWを増加させてから、その新しいレベルのボーナスを加算する。



例：



```text

FLOW 2の状態

→ DOUBLE成立

→ FLOW 3へ上昇

→ ラップボーナス+15

```

\# 15. FLOW段階報酬



1周の中で初めてFLOW 3とFLOW 5へ到達したとき、追加報酬を得る。



| FLOW到達 | 報酬         |

| -----: | ---------- |

| FLOW 3 | DICE\_ADD\_1 |

| FLOW 5 | DICE\_KEEP  |



状態：



```text

flow\_reward\_3\_claimed\_this\_lap

flow\_reward\_5\_claimed\_this\_lap

```



周回ゲート通過時にリセットする。



FLOWが低下し、同じ周回中に再到達しても二重取得しない。



段階報酬そのものは加速行動として数えない。

\# 18. ラップスコア



ラップスコアは、周回ポイントとは別に計算する。



用途：



\* 自己ベスト

\* ステージ記録

\* 将来のランキング

\* 週替わりチャレンジ



計算式：



```text

ラップスコア

= floor((100 + ラップボーナス + 速度ボーナス) × クリーン倍率)

\- デメリット減点

```



最低値：



```text

MIN\_LAP\_SCORE = 0

```



\## 18.1 速度ボーナス



90マス盤面の初期値。



| 1周のロール数 | 速度ボーナス |

| ------: | -----: |

|    10以下 |   +100 |

|   11〜13 |    +60 |

|   14〜16 |    +30 |

|   17〜20 |    +10 |

|    21以上 |      0 |



ワープや移動アイテムを使った高速周回を評価する。



\## 18.2 デメリット減点



実際に受けたデメリット1回につき：



```text

\-25

```



周回ポイントには適用しない。

\# 19. 周回完了処理



周回ゲート通過時の処理順：



1\. 周回数を1増加

2\. クリーン成立を判定

3\. クリーン連続数を更新

4\. クリーン倍率を決定

5\. 周回ポイントを計算

6\. ラップスコアを計算

7\. 累積周回ポイントへ加算

8\. 自己ベストを更新

9\. クリーン段階報酬を付与

10\. 既存の15コインを付与

11\. 周回スタンプと旅メモを追加

12\. ショップ商品を更新

13\. ボス状態を更新

14\. 次周用ラップ状態を初期化

15\. 自動保存

16\. 短い周回結果を表示



FLOWと現在ダイス数は周回ゲートでリセットしない。



周回をまたいで加速状態を持ち越せる。

\# 22. エンドレス進行



以下の状況でも旅は終了しない。



\* ボスをゲットした

\* 累積ポイント報酬を受け取った

\* クリーン連続5へ到達した

\* FLOW MAXへ到達した

\* 100周を超えた



ボスをゲットした場合は、次の個体を生成して同じ旅を継続する。



プレイヤーが「旅を休む」を選ぶまで、周回を続けられる。



アプリ終了後も同じ状態から再開できる。

\# 23. ボスの位置づけ



ボス交流は周回テンポを止めない。



基本方針：



\* 交流は5〜10秒

\* 選択肢は最大2〜3個

\* 長い会話は行わない

\* ボスはラップボーナスやアイテムを与える

\* ゲットしても旅を終了しない



ボス遭遇時：



```text

ラップボーナス +10

```



ボスゲット時：



```text

ラップボーナス +30

```



ゲット後の報酬候補：



\* DICE\_ADD\_1

\* DICE\_KEEP

\* 大きな旅箱

\* ポストカード

\* 次個体の気配+1

\* 点滅させない

\* 現在地表示でもミニマップでも確認可能



\## 24.2 メリットマス



\* 青緑、金、淡い水色

\* 追加ダイスや前進を連想できるアイコン

\* 危険マスより視認性を少し弱くする



\## 24.3 到達可能範囲

\# 32. テスト項目



1\. どの周回でも最低100ポイントを得る

2\. デメリットで累積周回ポイントが減らない

3\. ラップボーナスが正しく加算される

4\. クリーン時だけ倍率が適用される

5\. クリーン失敗時に連続数が1段階だけ下がる

6\. アイテムで完全無効化した場合、クリーンを維持する

7\. 危険マス突破成功でクリーンを維持する

8\. 危険マス失敗でクリーンを失う

9\. 加速行動が1ターンに1回だけFLOWを増やす

10\. 非加速ターンでFLOWが1だけ下がる

11\. デメリット適用時にFLOWが0になる

12\. FLOW 3報酬が同一周回で一度だけ発生する

13\. FLOW 5報酬が同一周回で一度だけ発生する

14\. 周回後もFLOWを持ち越す

15\. 周回後も現在ダイス数を持ち越す

16\. DOUBLEが周回ボーナスとFLOWへ反映される

17\. 全3ダイス役が正しく反映される

18\. ALL EVENガードが一度だけ働く

19\. ワープ移動先のマス効果が発動しない

20\. ぴったり切符で危険マスを避けられる

21\. ラップスコアと周回ポイントが別計算になる

22\. 速度ボーナス境界が正しい

23\. 周回報酬が二重付与されない

24\. アプリ中断後もラップ状態を復元できる

25\. 100周後も報酬テーブルが継続する

26\. ボスゲット後も旅を継続できる

27\. M3ボス回帰がPASSする

28\. M4Aイベント回帰がPASSする

29\. 1／2／3／5ダイス回帰がPASSする

30\. Goal checkerとJudge監査が正常に完了する

\# 33. 完了条件



\* 周回するたびに必ず進行を感じられる

\* デメリットを避ける意味が明確である

\* 失敗してもプレイ時間が無駄にならない

\* 1ダイス目押しが危険回避に役立つ

\* 追加ダイスと移動アイテムが周回速度を高める

\* メリットを連続させるとFLOWが育つ

\* クリーンラップ連続で倍率が成長する

\* 危険マスへ止まっても、目押しで突破を狙える

\* 周回ポイントとラップスコアが異なる遊び方を支える

\* ボスやイベントが周回テンポを大きく止めない

\* 100周後もプレイと報酬が継続する

\* 「あと一周だけ」が自然に発生する
