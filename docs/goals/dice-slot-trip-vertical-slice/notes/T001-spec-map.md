# T001 仕様マップ

- 正本: `Dice_Slot_Trip_仕様書.md` v0.1。
- MVP舞台: 砂時計のカイロ、約90マスの閉ループ、眠そうなスフィンクス。
- 第一スライス: M0〜M2。縦画面、画面遷移、Path2D / Curve2D盤面、3ダイス、役、周回、主要着地マス、1/5ダイス。
- 主要受け入れ: 20回以上連続ロールして位置破綻なし、89→0周回、移動中の二重入力禁止、全役をデバッグ再現、5ダイスは選択した3個だけを評価。
- 面白さの安全な補強: 一行旅メモ、気配に応じた環境変化、周回スタンプ、着地音のリズム、ボスが前回交流を覚える短い台詞。
- ビジュアル基準: 低彩度の砂色・淡い空色・青緑、旅手帳・古地図・絵本・ジオラマの中間。生成画像内の文字は利用せずGodot UIで重ねる。
- 現時点の環境差分: 仕様はGodot 4.7 stableだが、Godot MCPの報告は4.6.1 stable。4.7専用APIを避け、最終4.7確認を別ゲートにする。
