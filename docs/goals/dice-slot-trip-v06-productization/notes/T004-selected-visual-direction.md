# T004 — Selected Product Visual Direction

## Selection

骨格はB `Explorer's Atlas`。通常画面は中央の大きな地図、固定ダイストレイ、簡潔な上部HUD、明確な本線・バイパス・円環、低刺激な長時間向け色調、読みやすいタイムと自己ベストを採用する。

## Aから借りる移動感

- キャラクター周辺だけ地図が小さな世界として立ち上がる。
- 現在地付近のマスがわずかに盛り上がる。
- キャラクターは軌跡だけでなく、1マスずつ実際に前へ跳ぶ。
- カメラは現在地を柔らかく追い、進行方向の先を見せる。
- 遠いエリアは平面地図のまま、現在地周辺だけミニチュア化する。
- 全面3D化せず、制作コストとAndroid性能を抑える。

## Cから借りる山場

金色の門、ランタン、夜景、発光経路は常時使わず、次の山場に限定する。

- ボスゲート
- 夜の周回
- 特殊な円環
- 10周ごとの強化ボス
- TRIPLE成立時
- 自己ベスト更新直前

通常時から光らせず、頑張ったときだけ夜景が目を開く構成にする。

## Required Revision to B

- 現在地周辺を大きく表示し、全32マス常時表示を必須にしない。通常はキャラクター中心に先の8〜12マスを見せる。
- キャラクターを1.3〜1.5倍にして主役として読ませる。
- 地図全体固定ではなく、カメラがキャラクターを緩やかに追従する。
- 1マスずつ実移動し、軌跡だけが伸びる表現にしない。
- スロット3枠とサイコロを近づけ、出目と履歴の視線をまとめる。
- 出目確定前のスロット発光は呼吸する程度に抑える。
- マス番号を常時すべて表示せず、種類は記号・色、番号は現在地付近だけ表示する案を採用候補とする。

## Product Principle

「地図を開くと、歩いている場所だけが小さな世界として立ち上がる。」

This selected direction supersedes choosing A, B, or C unchanged. The next visual target must be a revised B implementing these conditions before T005 UI integration.
