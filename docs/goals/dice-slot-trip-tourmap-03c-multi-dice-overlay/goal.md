# TOURMAP-03C マップ上2／3／5ダイス

## Outcome

Tourism Map表示中、2／3／5個のCanvas 2.5Dダイスが下部トレイから同じLanding Zoneへ飛び出し、左から順に早止め・自動停止し、役と移動距離が読める結果保持を経て旅道具へ戻る。

## Non-negotiables

- Classic表示は既存トレイ演出を維持する。
- 1ダイスの03A挙動、出目、早止め、役、保存、移動、効果の正本を変更しない。
- Overlayはゲーム結果を所有せず、既存Main／DiceLogicへ値を渡すだけにする。
- 2個は横並び、3個は横一列、5個は3＋2の二段配置とする。
- 5個ロール中は選択前に到達先を誤って確定・強調しない。
- マップ上の現在地、危険、名所、到達可能範囲、景観中心を覆わない。
- 既存Audio pool、Canvas 2.5D、Android縦画面、保存トランザクションを維持する。

## Acceptance

- Tourism 1／2／3／5ダイスが同じ基盤で投射、停止、結果保持、帰還する。
- 2／3ダイスは合計値で到達先を強調し、5ダイスは3個選択後にのみ移動へ進む。
- 早止めは一回だけ有効で、ロール中の盤面入力は遮断される。
- 360×640、90→01、20回連続1ダイス、複数ダイスのnode／audio増殖なしがPASSする。
- 03A、03B、M3、M4A、CLEAN、RISK、LAP、LANDMARK回帰とGoal checker、Judge監査がPASSする。
