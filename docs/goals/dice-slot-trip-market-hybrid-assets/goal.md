# TOURMAP-02A 市場地区ハイブリッド素材実験

## Outcome

市場地区のTourismMapViewへ、単一のCC0無料アセットパックから選んだ道・石・壺・木箱・植物などの反復部品を加え、既存の香辛料市場Lv0〜3を主役のまま保つ。360×640で「素材集の寄せ集め」ではなく、一つの観光地図として見えることを確認する。

## Scope

- Kenney Sketch Desertを第一候補として公式ライセンスと内容を監査する。
- 採用時は `third_party/` に原本・LICENSE・source URL・attributionメモを保存する。
- 市場地区にだけ少数の装飾を追加し、共通の色調・影・彩度で既存景観へなじませる。
- Classic、90マス、ゲームロジック、セーブ、ダイス、他地区は変更しない。
- 360×640の比較キャプチャと回帰QAを残す。

## Non-goals

- 複数の無料パックを混在させない。
- GPT-Image 2で90マス分の小物を量産しない。
- TOURMAP-03のマップ上ダイス、TOURMAP-04カメラ、TOURMAP-05 FLOWを混ぜない。
- 香辛料市場Lv0〜3を置き換えない。

## Acceptance

- ライセンスと出典がリポジトリ内で追跡できる。
- 香辛料市場が視線の主役のまま、道と周辺に観光地らしい密度が増える。
- 現在地、到達範囲、危険、名所の判読性を損なわない。
- 360×640でクリップ・黒欠落・入力干渉がない。
- TOURMAP、CLEAN、LAP/LANDMARK、RISK、M3、M4A、save/dice回帰がPASSする。
