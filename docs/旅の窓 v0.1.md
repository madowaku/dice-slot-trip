よし、\*\*まずは課金口を一切開けない「旅の窓 v0.1」\*\*で組み込もう。🌍



採用するのはGoogle Maps Platform APIではなく、\*\*Google Maps URLsを外部起動する方式\*\*。これはAPIキー不要で、AndroidではGoogle Mapsアプリがあればアプリ、なければブラウザで開く。URLには`api=1`が必須。(\[Google for Developers]\[1])



\## 無料版の完成イメージ



名所カードに発展段階に応じたボタンを追加する。



| 名所発展  | 解禁            |

| ----- | ------------- |

| Lv1   | 📍 地図で見る      |

| Lv2   | 🛰️ 空から見る     |

| Lv3   | 🧭 360°で歩く    |

| 未対応地点 | 360°ボタンは表示しない |



「空から見る」はGoogle Earthを直接組み込まず、Google Mapsの衛星表示を使う。

「360°で歩く」はStreet Viewが実際に存在する名所だけ有効化する。



Street Viewの`viewpoint`は指定座標に最も近いパノラマへ接続するため、画像更新によって表示地点が少し変わる場合がある。各名所を実機で確認してから有効化するのが安全。(\[Google for Developers]\[2])

\## 3. 共通サービス



`travel\_window\_service.gd`



```gdscript

class\_name TravelWindowService

extends RefCounted



const MAPS\_SEARCH\_BASE := "https://www.google.com/maps/search/?api=1"

const MAPS\_VIEW\_BASE := "https://www.google.com/maps/@?api=1"





static func open\_place\_search(place\_query: String) -> Error:

&#x09;if place\_query.strip\_edges().is\_empty():

&#x09;	push\_warning("TravelWindow: place\_query is empty.")

&#x09;	return ERR\_INVALID\_PARAMETER



&#x09;var encoded\_query := place\_query.strip\_edges().uri\_encode()

&#x09;var url := "%s\&query=%s\&utm\_source=dice\_slot\_trip\&utm\_campaign=landmark\_view" % \[

&#x09;	MAPS\_SEARCH\_BASE,

&#x09;	encoded\_query

&#x09;]



&#x09;return OS.shell\_open(url)





static func open\_satellite(

&#x09;latitude: float,

&#x09;longitude: float,

&#x09;zoom: int = 17

) -> Error:

&#x09;if not \_is\_valid\_coordinate(latitude, longitude):

&#x09;	push\_warning("TravelWindow: invalid coordinates.")

&#x09;	return ERR\_INVALID\_PARAMETER



&#x09;var coordinate := ("%0.6f,%0.6f" % \[latitude, longitude]).uri\_encode()

&#x09;var safe\_zoom := clampi(zoom, 1, 21)



&#x09;var url := (

&#x09;	"%s\&map\_action=map\&center=%s\&zoom=%d\&basemap=satellite"

&#x09;	% \[MAPS\_VIEW\_BASE, coordinate, safe\_zoom]

&#x09;)



&#x09;return OS.shell\_open(url)





static func open\_street\_view(

&#x09;latitude: float,

&#x09;longitude: float,

&#x09;heading: float = 0.0,

&#x09;pitch: float = 0.0,

&#x09;fov: float = 80.0

) -> Error:

&#x09;if not \_is\_valid\_coordinate(latitude, longitude):

&#x09;	push\_warning("TravelWindow: invalid coordinates.")

&#x09;	return ERR\_INVALID\_PARAMETER



&#x09;var coordinate := ("%0.6f,%0.6f" % \[latitude, longitude]).uri\_encode()

&#x09;var safe\_heading := clampf(heading, -180.0, 360.0)

&#x09;var safe\_pitch := clampf(pitch, -90.0, 90.0)

&#x09;var safe\_fov := clampf(fov, 10.0, 100.0)



&#x09;var url := (

&#x09;	"%s\&map\_action=pano\&viewpoint=%s\&heading=%0.1f\&pitch=%0.1f\&fov=%0.1f"

&#x09;	% \[

&#x09;		MAPS\_VIEW\_BASE,

&#x09;		coordinate,

&#x09;		safe\_heading,

&#x09;		safe\_pitch,

&#x09;		safe\_fov

&#x09;	]

&#x09;)



&#x09;return OS.shell\_open(url)





static func \_is\_valid\_coordinate(latitude: float, longitude: float) -> bool:

&#x09;return (

&#x09;	latitude >= -90.0

&#x09;	and latitude <= 90.0

&#x09;	and longitude >= -180.0

&#x09;	and longitude <= 180.0

&#x09;)

```



Godotの`OS.shell\_open()`はURLを端末側の適切なアプリで開き、Android、Windowsなどに対応している。URL内の文字は`String.uri\_encode()`でエンコードするのが公式推奨。(\[Godot Engine documentation]\[3])

\## 課金防止の境界線



v0.1では以下を使わない。



\* Maps SDK for Android

\* Maps JavaScript API

\* Street View Static API

\* Street View Tiles

\* Places API

\* APIキー

\* Google Cloudの課金アカウント



Google Maps Platformの埋め込みAPI群は従量課金制で、現在はサービスごとに月間無料枠が設定されている。今回のMaps URLs方式とは別物なので、将来アプリ内埋め込みを検討するときだけ新しいスライスとして扱う。(\[Google for Developers]\[4])



これなら、\*\*ゲームのカイロを育てる → 本物のカイロをのぞく → またゲームへ帰って周回する\*\*という、現実とイラスト世界を往復する小さな窓が、APIキーなしで入れられるね。
