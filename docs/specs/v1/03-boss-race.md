# 反転ダイス・ボスレース v1.0

> Status: Normative
> Synced: 2026-07-22

**「各周の最後に一発勝負」「ボスはプレイヤーの裏目で進む」**にすると、ボス戦が再挑戦用の関所ではなく、周回ごとの成績イベントになる。
## ボス戦の確定方向
各周の終点で、短いボスレースを行う。
勝っても負けても結果を記録し、次の周へ進む。
ボスの出目は、プレイヤーが止めた面の裏側とする。
### 裏目ルール
一般的な6面ダイスの向かい合う面は、合計7になる。

実装上は単純に、
boss_roll = 7 - player_roll
で決定できる。
このルール、かなり美しい。ボスが裏で勝手にサイコロを振らないので、負けても理由が透明になる。
## 面白くなる理由
### 一投がそのまま綱引きになる
プレイヤーが6を出せば、
猫：6進む
ボス：1進む
プレイヤーが1なら、
猫：1進む
ボス：6進む
高い目を当てる価値が、通常の移動以上に大きくなる。
ただし、単に毎回6を狙うだけでは薄いので、
- BOOSTへ正確に止まりたい
- SANDを避けたい
- 3ROLL SLOTのTRIPLEを狙いたい
- 探検猫のスキルをためたい
という別の目的をぶつける。
例えば、
プレイヤーの前方2：BOOST +3
スロット：[2][2][ ]
なら2を狙うと、
- 通常移動2
- BOOSTで追加3
- TRIPLE成立
- ボスは5進む
となる。
表面的には低い目だけれど、総合的には強い。ここに判断が生まれる。
## 各周のボス戦
### 勝敗はその周だけで完結
- 勝利：その周のボス勝利記録を獲得
- 敗北：敗北記録を付けて次周へ
- 本編を巻き戻さない
- ボスレースをやり直さない
- 所持品やタイムも巻き戻さない
例えば3周のステージなら、結果はこうなる。
第1周　WIN
第2周　LOSE
第3周　WIN
最終結果：2勝1敗
クリアそのものと、ボス勝敗を分離する。
### 勝利報酬
勝利で強いアイテムを与えると、勝った人が次も勝ちやすくなる。雪だるま式の格差は避けたい。
最初は記録・スコア寄りがよい。
- ボス勲章＋1
- クリアスコア加算
- 最終評価へ反映
- 旅行アルバムのスタンプ
- タイムから数秒ボーナス控除
敗北時は資源を奪わず、単にその周のボーナスを逃す。

通常ステージの最後に、短い決闘を差し込む構造。
### 最初のマス構成
初期版は3種だけで十分。
- NORMAL：効果なし
- BOOST：停止すると追加＋2
- SAND：停止すると次の移動－1
猫とボスは同じ配置の並走レーンを走る。
猫　　START ─ N ─ BOOST ─ N ─ SAND ─ N ─ BOOST ─ GOAL
ボス　START ─ N ─ BOOST ─ N ─ SAND ─ N ─ BOOST ─ GOAL
ボスにもBOOSTとSANDを適用する。ルールが左右対称なので、見ていて理解しやすい。
## 一投の処理順
1. プレイヤーがサイコロを止める
2. プレイヤー出目を固定
3. ボスの裏目を表示
4. 3ROLL SLOTへ転写
5. 役を判定
6. 両者の移動量を確定
7. 猫とボスが進む
8. 停止マス効果を処理
9. 勝敗を確認
表示例：
PLAYER  5
BOSS    2
ボス用に別の回転ダイスを出す必要はない。
プレイヤーのダイスが止まったあと、上部のボス側へ裏目がパチンと現れれば十分。
視線負荷も減る。
## 移動は論理的に同時
見た目では100〜150msほどずらしてもよいけれど、勝敗計算は同時に行う。
両者が同じターンでゴールへ到達した場合は、各移動を1歩ずつ進めたものとして、先にゴール線へ触れた側が勝つ。
例：
猫：ゴールまで2、出目5
ボス：ゴールまで1、出目2
ボスは1歩目でゴールするため、ボス勝利。
完全に同じ歩数で同時到着した場合は、プレイヤー勝利でよいと思う。毎周行うイベントなので、僅差判定は遊ぶ側へ寄せた方が気持ちよい。
## 3ROLL SLOTは維持
ボス戦でも通常と同じスロットを使う。
道中終了時：[4][4][ ]
最初のボス投射で、
- 4を狙うとTRIPLE
- 6を狙うと自分6、ボス1
- 2を狙うとBOOST停止
という三つ巴の判断が起きる。
ボス戦だけ別ルールへ切り替えず、道中の判断をそのまま持ち込めるのが強い。
## 探検猫との相性
ピンポイントを使うと、
- 高い目を狙って相手を遅くする
- BOOSTへ正確に止める
- SANDを避ける
- TRIPLEを完成させる
を助けられる。
裏目方式では、スキル成功が二重に効く。
5を狙って成功
→ 猫5
→ ボス2
それでも自動成功ではないので、プレイヤー本人の腕前は残る。
## 注意点
### 高い数字だけが強くなりすぎる
裏目方式では6が非常に強い。
そこでボスコースには、低い出目を選ぶ理由を必ず配置する。
- 2でBOOSTへ正確停止
- 3でSANDを回避
- 4でSTRAIGHT成立
- 1でTRIPLE成立
最低でも、各ターンに「最大値以外の有力候補」が一つ見える盤面にする。
### ボスへ隠れ補正を入れない
- ゴール前だけ強い出目
- 負けていると移動増加
- プレイヤーが勝ちそうになるとBOOST発動
のような裏補正は不要。
ボスの出目が完全にプレイヤーの裏目だからこそ、このルールは信頼できる。
## 確定案
### 反転ダイス・ボスレース
- 各周の最後に12〜14マスの短い並走レース
- 勝敗にかかわらず次周へ進む
- ボス戦のやり直しなし
- プレイヤー出目をxとするとボスは7-x
- 猫とボスは同じ配置のレーンを走る
- 3ROLL SLOT、探検猫スキル、道中状態を持ち込む
- 勝利数は最終評価、記録、スタンプへ反映
- 完全同着だけプレイヤー勝利
この方式なら、ボスは別ゲームではなく、普段の一投がそのまま決闘へ変わる鏡面レースになる。かなりDice Slot Tripらしいボスルールだと思う。
