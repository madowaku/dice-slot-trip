# 設計整理の背景

> Source: https://docs.google.com/document/d/1tk-zU_aOoHfa4ooO_wKzmblM-2WxouacFXtha4WL0Zo/edit
> Status: Context — 検討経緯。確定仕様と競合する場合は後者を優先
> Synced: 2026-07-22

よし、ここで一度、仕様の砂嵐を止めてゲームそのものの芯を見直そう。
今までの会話、家族フィードバック、プレイ動画から見ると、機能はかなり育ったけれど、「何が一番面白いゲームなのか」はまだ最終確定前だと思う。
## まず結論
Dice Slot Tripの最も強い核は、
先のマスを見て欲しい出目を狙い、その結果が移動と3投スロットの両方へ影響すること
だと思う。
旅、ボス、アイテム、円環は、この面白さを引き立てる脇役。
脇役が舞台中央へ押し寄せると、ただ情報量の多いすごろくになってしまう。
設計上の合言葉はこれがよさそう。
一投につき、二つの悩み。
「どこへ止まりたいか」と「3投目に何を作りたいか」。
## 1. 今のすごろくゲームの面白さを再確認
### 面白さの第1層：目押し技術
プレイヤーは運任せではなく、欲しい目を狙える。
- 先のRISKを避ける
- ITEMへ止まる
- 円環のEXITへ正確に止まる
- バイパス入口へ入る
- スロットのPAIRやTRIPLEを完成させる
何周も遊ぶほど成功率が上がる。
ここがテトリスや数独に近い、プレイヤー本人が上達する部分。
これは絶対に残す。
### 面白さの第2層：一つの出目に二つの意味がある
例えば現在の履歴が、
[6][6][＿]
前方4マスに円環出口がある。
- 4を狙えば脱出
- 6を狙えばTRIPLE
この迷いがDice Slot Trip固有の面白さになりうる。
ただ6を出すだけでも、ただ役を作るだけでもない。
位置と履歴が喧嘩する瞬間が面白い。
### 面白さの第3層：自分の走り方を作る
- 安全な本線
- 速いバイパス
- 報酬の多い円環
- アイテムを集めてボスへ
- 準備を捨てて自己ベストを狙う
同じステージでも、「今日は安全に」「今日は最速で」が変わる。
ただし、ルート選択が面白くなるには、ルートごとの差が明確である必要がある。
- 本線：安全
- バイパス：速いが危険
- 円環：遅いが報酬と技術勝負
この三角形は残したい。
## 2. まだ面白さが証明されていない部分
ここは一度、冷静に分けよう。
### 3投スロット
発想はかなり良い。ただし、
PAIRやTRIPLEが完成した結果、本当に嬉しいか
はまだ十分に検証されていない。
画面に「PAIR！」と出るだけでは弱い。
スロットは演出ではなく、プレイ方針を変える報酬装置である必要がある。
### ボス戦
周回末に山場がある構成は良い。ただ、
- 合計値が高い方が勝ち
- HPを3回削る
だけだと、道中より単純になる危険がある。
道中で磨いた「狙った出目へ止める技術」が、ボスでも別の形で必要になるべき。
### アイテムとスキル
便利そうだけれど、増やしすぎると目押しよりメニュー操作が主役になる。
特に、
- 出目補正
- 速度低下
- ダメージ無効
- スロット保持
- ボス盾
- 追加ダイス
が全部同時にあると、何が勝因だったのか分かりにくい。
最初は少数に絞って、「いつ使うか」が面白いかを見るべき。

## 4. 新ルールの壁打ち
ここからが本丸。
### 案A：3投スロットを「旅の選択権」にする
3投完成時の役を、単なる報酬ではなくルート操作へ接続する。
#### PAIR
小さな戦術報酬。
- 次のRISKを軽減
- コイン獲得
- アイテムゲージ上昇
- 出目補正札を獲得
#### TRIPLE
大きな報酬。
- ボス用追加ダイス
- バイパスの強制開通
- 円環から即脱出
- ボス弱点を1つ破壊
#### STRAIGHT
ルート操作報酬。
- 次の分岐を自由選択
- ワープ先を選択
- 先6マスを一度だけ並べ替える
- 追加移動せず、目的マスへ正確停止
この形なら、役ごとに用途が違う。
- PAIR：安定
- TRIPLE：爆発力
- STRAIGHT：経路制御
かなり相性が良い。
### 案B：マスを踏むだけでなく「通過」も意味を持たせる
現在は停止したマスだけが重要になりやすい。
通過時にも弱い効果を持たせる案。
例：
- COIN通過：1枚、停止：3枚
- REST通過：スキルゲージ、停止：HP回復
- ITEM通過：中身を確認、停止：獲得
- RISK通過：無害、停止：罰
- EVENT通過：予告、停止：イベント
これにより、大きな出目にも意味が残る。
ただし処理が増えすぎる危険があるので、最初はCOINだけ通過取得でもよい。
### 案C：RISKを罰ではなく技術課題にする
単にHP－1では、止まった瞬間に終わる。
RISKへ止まったら、短い目押しチャレンジを発生させる。
例：
- 3を狙えれば回避
- 偶数なら無傷
- タイミング帯の中央で止めれば成功
- 現在のスロットと同じ数字なら反撃
上手い人はRISKを近道として利用できる。
これなら「危険だけど技術で抜ける」という本作らしさが出る。ただし毎回やると重いので、強いRISKだけに限定する。
### 案D：ボスを「出目弱点パズル」にする
ボスには毎ラウンド、3つの弱点を表示する。
例：
弱点：[2][5][PAIR]
3投で、
- 2を含む
- 5を含む
- PAIRを作る
のうち達成した数だけダメージ。
別のボスなら、
弱点：[偶数][合計12以上][TRIPLE]
になる。
これなら道中と同じ技術を使いながら、ボスごとに攻略が変わる。
単純な合計値勝負より、かなりDice Slot Tripらしい。
## 5. おすすめする新しい基本ルール
今の本命はこれ。
### 一投の流れ
- 前方6マスを見る
- スロット履歴を見る
- 欲しい目を決める
- サイコロを投げる
- 目押しで止める
- 出目を見る
- スロットへ保存
- キャラクターが移動
- 停止マスを処理
### 3投目
役が完成し、以下のどれかが発動する。
- PAIR：小さな防御・資源
- STRAIGHT：ルート操作
- TRIPLE：ボス用の強力資源
### ステージ内
- 本線：安定
- バイパス：高速
- 円環：報酬と目押し課題
- ボス：出目弱点パズル
### 記録
- クリアタイム
- 投射数
- 目押し精度
- 円環脱出精度
- ボス弱点達成率
- PAIR / STRAIGHT / TRIPLE数
これなら、ゲーム中のほぼすべてが「目を狙う技術」へ戻ってくる。
## 6. Dice Slot Tripの配合比
現時点では、ゲームの性格をこう置くのがよさそう。
- 技術型目押しすごろく：70％
- ルート選択・軽量ローグライト：20％
- 旅情・キャラクター・ボス演出：10％
旅情を薄くするという意味ではない。
画面の魅力は旅で作り、判断の魅力は目押しで作る。
土台は競技、衣装は旅行。そんな配合。
